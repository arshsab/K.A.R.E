How it works

Kare takes advantage of Github's starring and watching features to gather data on user interest in repositories. It analyzes this data to find repositories that are similar. Unfortunately it isn't able to analyze every repository on Github so instead it has information for the top repos as ranked by number of stars to ensure that repos we have enough data for are in our database.
